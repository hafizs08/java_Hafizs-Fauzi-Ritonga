(JWT) JSON Web Token adalah standar Internet yang diusulkan untuk membuat data dengan tanda tangan opsional dan/atau enkripsi opsional yang muatannya menampung JSON yang menegaskan sejumlah klaim. 

jwt spring boot mengunakan [JWT](https://jwt.io/) untuk menyimpan data yang dibutuhkan untuk mengakses aplikasi.
